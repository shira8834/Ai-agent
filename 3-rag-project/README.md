🎨 Project: Multi-Lingual Color Expert RAG
📌 סקירה כללית
פרויקט זה מיישם מערכת RAG (Retrieval-Augmented Generation) חכמה המתמחה בעולם הצבעים ושילוביהם. המערכת סורקת מאגר ידע מובנה (JSON) המכיל הנחיות, חוקי ערבוב ותיאוריות צבע, ומאפשרת למשתמש לתחקר את המידע בצורה טבעית מבלי שהמודל יחרוג מהעובדות הקיימות במאגר.

🎯 מטרות הפרויקט
דיוק עובדתי: מניעת "הזיות" (Hallucinations) של ה-AI על ידי הגבלת התשובות להקשר (Context) בלבד.

תמיכה רב-לשונית: שימוש ב-Embeddings מתקדמים התומכים בעברית ובאנגלית בצורה שווה.

ממשק אינטראקטיבי: הנגשת הידע הטכני דרך ממשק צ'אט ידידותי למשתמש.

🛠️ טכנולוגיות בשימוש
Framework: LangChain (ניהול השרשרת וה-Retriever)

LLM & Embeddings: Cohere (Command-R, Multilingual-v3)

Vector DB: ChromaDB (אחסון וחיפוש וקטורי מקומי)

UI: Streamlit

🧩 מבנה המערכת (Pipeline)
המערכת בנויה משני שלבים עיקריים:

שלב ההכנה (Pre-processing):

טעינת colors_data.json.

יצירת ייצוג מתמטי (Embeddings) לכל שאלה ותשובה במאגר.

אינדוקס הנתונים ב-ChromaDB לחיפוש מהיר.

שלב ההרצה (Runtime):

קבלת שאלה מהמשתמש דרך Streamlit.

שליפת המידע הרלוונטי ביותר (Top-K) מתוך בסיס הנתונים הווקטורי.

בניית Prompt "קשוח" המכריח את המודל להשתמש רק במידע שנשלף.

ייצור תשובה סופית והצגתה בצ'אט.

📂 מבנה קבצי הפרויקט
logic.py: "המנוע" של המערכת - הגדרת ה-Chain, ה-Retriever והחיבור ל-Cohere.

app.py: ממשק המשתמש ב-Streamlit וניהול היסטוריית הצ'אט.

colors_data.json: מאגר המידע המובנה המכיל את כל הידע על הצבעים.

.env: ניהול מפתחות API (לא מועלה ל-Git).

🚀 הוראות הרצה
התקנת ספריות: ```bash
pip install langchain-cohere langchain-chroma streamlit python-dotenv

הגדרת מפתח API: יש ליצור קובץ .env ולהזין COHERE_API_KEY=your_key.

הרצת הממשק:

Bash

streamlit run app.py
